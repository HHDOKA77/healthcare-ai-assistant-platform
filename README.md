# 🩺 HealthCare AI Assistant Platform

An AI-powered healthcare assistant developed using Amazon AWS PartyRock and Generative AI.

## 📌 Overview

The HealthCare AI Assistant Platform helps users better understand their health conditions by analyzing symptoms and providing personalized educational guidance. It is designed to improve health awareness and support informed healthcare decisions.

**Key Features**
- 👤 Patient information collection
- 🩺 Medical history review
- 💊 Medication guidance
- 🤒 AI-powered symptom analysis
- 📚 Health education
- 🚨 Alerts users when professional medical care may be needed

## ⚙️ Technologies Used
- Amazon AWS PartyRock
- Generative AI
- Large Language Models (LLMs)
- Prompt Engineering
- Natural Language Processing (NLP)

## ⚠️ Disclaimer
This application provides educational information only and does not replace professional medical advice or diagnosis.
